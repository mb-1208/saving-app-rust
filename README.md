# Saving App (Anchor)

## Prerequisites 

1. Install Rust, Anchor, Solana Tools, Node.js.

2. Install node_modules before run program:
    
    $ yarn / yarn install / npm i / npm install
       
## Generate 

1. Run

    ```shell
    $ anchor build
    ```

2. Run

    ```shell
    $ anchor deploy
    ```
    
3. Change declare_id with new program_id after deploy

4. Run

    ```shell
    $ anchor test
    ```
